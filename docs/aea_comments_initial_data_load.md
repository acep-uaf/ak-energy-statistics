Attached is an Excel file containing the initial data load. The data included represents UMR records that were created or modified between January 1, 2016, and May 31, 2026.

While the intent is to provide data starting from calendar year 2016, you may notice some records from earlier years. These are included because they were modified within the specified date range.

The file contains two worksheets:
Header data
Rate Line data
You can join the Header and Rate Line data using the ID column.

Starting in July, you will receive a monthly email on the 7th with any additions or modifications from the prior month. Please note that these updates may include changes to records that already exist in your system.

As part of your import process:
For Header data, use the ID field to determine if a record already exists.
For Rate Line data, use a combination of the ID field and the Line No field.
If a record already exists, update it; otherwise, add it as a new record.
