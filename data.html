Level 1 Data

Level 1 data is created by running basic data quality checks against the data and overwriting with NULL for values outside the boundaries. For instance, the column most_recent_fuel_purch_price requires values to be greater than $0.50 and less than $10.00. If a record contains a value of $0.20 or $50.00, the value is discarded. A list of records with out-of-bound values can be seen on Quality and Outliers
